# GSAT-7000words：高中英文參考詞彙表資料集

## 專案簡介

本專案致力於整理並提供數位化的「高中英文參考詞彙表」（俗稱 7000 單字）。此資料集根據大學入學考試中心（CEEC）公佈的清單進行編碼與整理，旨在為教育工作者、研究人員及開發者提供結構化的語言資料，適用於 108 課綱之學測與分科測驗準備。

## 資料特點

* 完整性：收錄 Level 1 至 Level 6 之核心詞彙。
* 結構化：資料以易於解析的格式儲存，方便進行資料分析或匯入資料庫。

## 資料格式與結構

資料主要存放在 data 目錄中，包含以下內容：
* 分級單字表：依據難易度分層的文字檔案。
* 彙整表：整合所有級別的結構化檔案。

舉例使用方式：
* 工具整合：將資料匯入至 Anki、Quizlet 或自建的學習系統中。
* 教育用途：作為製作講義或測驗卷的基礎字庫。
